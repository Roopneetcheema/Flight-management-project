# Flight Management System

## Description
This project is a Flight Management System built using Oracle SQL and Oracle APEX.

## Features
- Flight management
- Passenger management
- Seat booking system
- Luggage validation (trigger)
- VIP passenger handling
- Automated price calculation (function)

## Technologies Used
- Oracle SQL
- PL/SQL
- Oracle APEX

## How to Run
1. Run SQL Codes into Oracle SQL Workshop
   or
1. Import apex_app.sql into Oracle APEX (App Builder → Import)
2. Run application
